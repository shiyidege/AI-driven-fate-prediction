# 知命阁-AI算命项目
## 项目简介
本项目是基于大语言模型（LLM）工程能力搭建的算命类网站，核心依托 LLM API 调用、Prompt Engineering、知识库搭建、RAG 检索增强生成及系统评估优化等技术，通过导入周易、梅花易数、塔罗牌等传统命理书籍及历史算命案例数据，结合用户输入/选择的个人信息，生成定制化算命结果，为用户的感情、事业、学习、风水选择等场景提供专业指导与心理支持。

### 核心价值
- **心理价值**：肯定用户过往行为轨迹，输出未来行动方向建议，同时提供针对性心理抚慰；
- **实际价值**：基于用户提交信息进行科学分析，输出可落地的决策指引，助力事业、感情、学习等维度的发展。

### 创新亮点
1. **灵活计费模式**：采用 Token 按量计费，用户无需支付高额月/年会员费，准入门槛低；平台通过模型调用差价实现盈利，兼顾用户体验与商业收益；
2. **高响应保障**：采用「回答保底 + LLM 实时回答」机制，确保结果及时返回，提升用户体验；
3. **精准结果输出**：融合 Prompt Engineering、高质量知识库搭建、RAG 等技术，通过优质数据输入保障算命结果与行为建议的认可度；
4. **高度个性化**：支持用户自选信息输入方式（对话框/选项选择）、自定义调取基座模型（Qwen、DeepSeek、Gemini 等）；
5. **深度服务延伸**：在基础算命结果之上，新增趋利避害、逢凶化吉的具体行为建议。

## 技术方案
### 核心技术栈
| 模块         | 选型说明                                                                 |
|--------------|--------------------------------------------------------------------------|
| 前端（Web/App） | 适配多端的轻量化框架（Vue/React/Flutter）                                |
| 后端         | Python + FastAPI（或 Node.js + NestJS），兼顾 LLM 集成与异步高性能处理    |
| 数据库       | 主库：PostgreSQL/MySQL（存储用户、订单、Token 计费等核心数据）；<br>向量库：ChromaDB/Pinecone/Milvus（支撑 RAG 检索） |
| 第三方服务   | 基座模型 API：Gemini/GPT-4/Qwen/DeepSeek；<br>支付接口：Stripe/微信支付/支付宝 |

### 系统架构
整体采用分层架构设计，核心层级包括：
1. 前端层（Web/App）：用户交互入口，负责信息输入与结果展示；
2. API 网关/业务逻辑层：请求分发、Token 计费、权限校验等核心业务逻辑；
3. 知识库服务层：负责命理知识库的管理与 RAG 检索增强；
4. LLM Service 层：统一调用各类大模型 API，处理 Prompt 拼接与结果生成；
5. 数据层：业务数据与向量数据的存储、读写与管理。

### 实现难点与解决方案
- **核心难点**：输出准确、可信的算命结果，满足用户心理预期与实际参考需求；
- **解决方案**：构建高质量命理知识库（清洗周易/塔罗牌等书籍数据、历史案例），结合精细化 Prompt Engineering 优化生成逻辑，同时通过用户反馈持续迭代 RAG 检索策略。

## 目标与展望
### 比赛期 MVP 目标
完成可独立运行的算命网页与 APP 最小可行产品，实现核心功能：
- 用户信息输入（自定义输入/选项选择）；
- 多基座模型切换调用；
- Token 按量计费与支付；
- 算命结果生成与行为建议输出。

### 长期发展潜力
1. **技术迭代**：基于用户输入、反馈及结果数据，持续优化底层模型与算法，提升算命结果的指导性与用户认可度；
2. **盈利拓展**：接入广告 API，通过广告曝光、流量引流等方式增加收入来源；
3. **功能延伸**：新增求职辅导、恋爱建议、职场突破、个人精进等场景化功能，覆盖更广泛用户群体；
4. **流量运营**：通过投流、网赚合作、流量置换等方式扩大用户规模；
5. **个性化运营**：基于用户偏好分层（如姻缘/职场/学业），定向推荐定制化内容与服务。
